Etiquetas y tags

DOCTYPE html

Es un TAG que se usa al momento de configurar una página web, esto para saber en qué versión de HTML estamos trabajando, para que el navegador o editor de texto pueda procesar la información de una forma correcta.

html

Este TAG sirve para poder darle una buena estructura, y poder distribuir de buena manera el contenido de la página web y al igual que cada uno de las éticas se debe cerrar .

head

Es de las primeras dos partes de la estructura de una página web, muchos diseñadores de páginas web se confunden con la etiqueta de title, pero dentro de "head" se encuentra todo el contenido de una página web.

title

Como todo documento, o archivo tiene un título, pues igual las páginas web, esta etiqueta siempre va dentro de un head y se puede ver en las pestañas del navegador o en la barra de título.

body

Se usa para poder poner dentro del todo el contenido y diseño de una página web, por lo regular todo diseñador pone dentro del body los aspectos representativos de una página, y es una etiqueta fundamental para el diseño de páginas web.

h1

Es una etiqueta que nos sirve para indicar algunas palabras clave que se le puede dar al contenido de una página web, y no solo se usa un H1, si no que H2, H3 y H4 te pueden ayudar para resaltar algunas palabras que sean específicas o importantes en del tema principal.

p

Es una etiqueta para que sirve para poder distribuir bien los párrafos de la página.
