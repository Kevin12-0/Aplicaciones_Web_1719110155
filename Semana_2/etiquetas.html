Etiquetas <insert>,<select> y <textarea>

Etiqueta Descripcion Ejemplo de etiqueta
<input type="hidden"> esta etiqueta sirve para ocultar algunos de los elementos que están por ser creados, que están siendo creados, pero que aún no se quiere mostrar al Publio o para ocultar algún elemento que ya no es relevante dentó de la página, se considera un elemento de tipo booleano.

Nombre:

<input type="text"> este comando se utiliza para una entrada de texto, que permite al usuario insertar o editar una línea de texto regular, regularmente se usa este tipo de entrada, para solicitar lo que son, nombres, apellidos, descripciones breves, números de teléfono entre otras cosas.

Nombre:

<input type="search"> este comando e utiliza para buscar determinados, regularmente se estos tipos de inputs, están asosciados a cajas de texto, aunque algunas personas tienden a confundirlos con un text.

Buscar:

<input type="tel"> representa un campo que está asociado para los números de teléfono, y solo es para poner datos o números de manera regular, principalmente se adaptó este tipo de etiqueta por la gran variedad de números que existen en el mundo.

Telefono:

<input type="url"> se usa para rellenar, con alguna ruta del tipo url de manera absoluta.

URL:

<input type="email"> representa campos para una o más direcciones de correo electrónico, aunque principalmente solo se asocia con una sola dirección web, a los usuarios es permitido llenar el campo en formato texto regular.

Correo:

<input type="password"> representa un campo para las contraseñas, estos son similares a los campos de texto, solo con la diferencia que su valor se encuentra oculto y no permite a un usuario copiarlo actuando como una medida de seguridad.

Contraseña:

<input type="date"> se usa para insertar valores de fecha, una de las grandes ventajas de este tipo de comando es que por defult en navegador, convierte la fecha ingresada a un dato del tipo texto para ser registrado sin problemas.

Fecha:

<input type="month"> es un campo que solo es usado para la entrada de meses, al igual que con el de tipo date, este campo se convierte por default en un campo tipo text, al momento de completar y enviar algún formulario.

Mes:

<input type="week"> es un campo que nos permite tener entradas para semas del año, algunas veces no se ocupa este tipo de input y es reemplazado por un dato de tipo cadena.

semana:

<input type="time"> es un campo que se usa para representar la entrada de horas, aunque es muy difícil que sea reemplazado por algún otro tipo de comando para el ingreso de horas, este tipo de etiqueta solo permite entradas de números en horas del 0 al 23 y para los minutos del 0 al 60.

hora:

<input type="datatime-local"> representa un campo para la fecha hora sin información de una zona horaria, es muy útil para la recolección de datos de un lugar en específico.

Tiempo local:

<input type="number"> se usa para insertar datos de tipo numérico, los números negativos los representa con el guion - y los números decimales con un carácter del tipo "." seguido de los demás dígitos del número.

Numeros:

<input type="range"> se usa para insertar números dentro de algún rango, aunque no es importante seleccionar un valor exacto del rango algunas veces, muchos diseñadores los reemplazan por barras deslizadoras para poder seleccionar algún valor.

Volumen:

<input type="color"> es simplemente un campo para la entrada de un color, cada color es dado por cogido hexadecimal, comenzando con un # antes de cada código.

color:

<input type="checkbox"> se usa para representar dos campos para marcar o desmarcar, dos o más estados y es usado para la recolección de información que se usan para las preguntas de verdadero o falso, sexualidad, aceptar o rechazar entre muchos otros ejemplos.

checkbox: Perros
<input type="radio"> es algo parecido al checkbox, ya que sirve para opciones, aunque tu todos comparten un mismo valor de atributo, y juega un papel fundamental para las opciones.

radio: Masculino
<input type="file"> es para agregar archivos del computador, para ser subidos al servidor web, además de que cuenta con un atributo que se llama accept que se usa para encontrar los archivos que se buscan subir.

Archivo:
<input type="submit"> se representa a través de un botón, que envía los datos a un formulario correspondiente.

<input type="image"> representa un botón, pero a diferencia con los botones convencionales estos tipos de botones tienen una imagen, además de que cuando se le da clic a un botón de este tipo se hacen registro de los pixeles donde se le dé un clic.

<input type="reset"> como su nombre lo dice este formulario es para borrar los datos y regresar todos los campos del formulario a la forma original.

<input type="button"> se utiliza para crear datos, sin una acción definida asociado automáticamente, debes de darle una acción para que funcione, de lo contrario al presionarlo no ocurrirá nada.

<select multiple> se usa para que un usario pueda seleccionar varios tipos de datos de una lista, si es que estas están especificadas, de lo contrario si no están especificadas solo se pueden seleccionar una vez.
<select name> se usa para darle un nombre a algún elemento de control.
<select requiered> indica si alguna opción de selección esta desmarcada o si es necesaria para llenar algún formulario
<select size> se usa para representar una lista o un scroll como muchos lo conocen, para ser usado como número de líneas, que el control mostrara de una forma simultánea
